# Web Tracking: a library to analyize web browsing behavior.

## Install

1. Download repo

    ```bash
    git clone https://github.com/gesiscss/web_tracking.git
    ```

2. Open the repo

    ```bash
    cd web_tracking/
    ```

3. Install

    ```bash
    python setup.py install
    ```
## Analyses
